taurus.qt.qtgui.qwt5¶

Taurus Widget qwt5 (former plot) module¶

This module provides the Qwt5-based widgets previously provided by taurus.qt.qtgui.plot. Note that PyQwt5 module only works with Python2 and PyQt4 and is no longer supported, so taurus is moving to other modules for plotting (pyqtgraph, silx, ...)
